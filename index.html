React.js

The Future Of App Development

Some Basic Info About React:

-It Contains Javascript And HTML Both.

-It Has A Different Type Of Language JSX Comprised Of HTML And Javascript.

-It Has Several HTML Tags And Javascript Functions.

Declarative Language-

This Is A Language That Is Used In HTML Which Makes This Language Completely Easy As We Do Not Have To Describe How To Do Only We Have To Use The Tags In Which Automatically How Is Stored.

Imperative Language-

This Language Is Used In Javascript And Is A Language Which Is Difficult Than HTML But More Useful Than HTML. It Is Difficult Because In This We Have To Describe How Even For The Bouncing Of A Ball.

JSX-

This Is Also A Language Comprised Of HTML And Javascript. Have You Heard Of Facebook? It Is Made From JSX. JSX Was Founded By The Makers Of Facebook While They Were Making Facebook Through HTML They Mixed Both HTML And Javascript.
